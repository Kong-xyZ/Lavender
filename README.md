## 一、Lavener 是什么

Lavener 是一个 Gradle Plugin，提供一系列静态检测的能力，其目标主要是为了解决随着 APP 复杂度的提升而带来的性能、稳定性、包体积等一系列质量问题。

## 二、为什么以此命名？

Lavender 直译为薰衣草。薰衣草的香味能够帮助我们缓解压力、减少沮丧。

希望这个 Gradle Plugin 也能够帮助我们减少重复的劳动工作，使我们的工作更加轻松顺畅。
